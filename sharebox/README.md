# sharebox

プロジェクト全体の指示・報告の記録です。

## 用途

- PMが処理した全ての指示・報告がここに蓄積される
- チャットをクリアした後、`/restart` でここから経緯を復元できる
- プロジェクトの監査ログ・履歴として機能

## 管理者

PMのみがこのディレクトリを管理します。各担当は直接触りません。

## ファイル命名規則

```
YYYY-MM-DD-NNN-[type]-[from]-to-[to]-[title].md
```

例:
- `2026-01-22-001-instruction-pm-to-front-create-lp.md`
- `2026-01-22-002-report-front-to-pm-lp-complete.md`
